# Agenda Empresarial

`Desenvolvimento e Analise de Sistemas`

`Projeto: Desenvolvimento de Aplicação Web Front-End`

`Eixo 1`

Desenvolvimento de uma agenda online onde as informações e interações podem ser feitas com todos usuários de um determinado setor em uma determinada empresa, buscando  mais organização e diminuir o tempo de desenvolvimento de suas tarefas.
